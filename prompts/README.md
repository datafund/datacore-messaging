# Messaging Prompts

Reusable prompts for messaging module.

## Message Composition

### Professional Message
```
Write a professional message to [recipient] about [topic].
Keep it concise (2-3 sentences).
Include any relevant action items.
```

### Follow-up Message
```
Write a follow-up message regarding [original topic].
Reference the previous conversation.
Ask for status or next steps.
```

## AI Task Delegation

### Research Request
```
/msg @claude "Research [topic].
Focus on: [specific aspects]
Output to: [location]
Format: [literature note / summary / analysis]"
```

### Content Request
```
/msg @claude "Write [content type] about [topic].
Audience: [target audience]
Tone: [professional / casual / technical]
Length: [word count or sections]"
```
